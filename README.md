# sangam
Hackathon project, connecting students.

## Description
A networking chatbot where a user can be a student or a professor. A student can prompt the chatbot he/she wants to connect to people in the college with aligning interests and skillset. And a professor can prompt that he/she is working on a project and he wants to recruit RA/TA. The chatbot first lets the professor know what kind of tech stack could be most relevant to build the project. If the the professor approves, it then searches students' profiles and shows top 5 results, from most recommended to least recommended.
